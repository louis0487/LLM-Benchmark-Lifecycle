# The Lifecycle and Clustering of LLM Benchmarks – A Data Driven Analysis
## Project Information
For project information, technical approach and results, please see `docs/project.md`. 

## Repository Structure

- **`/src/`**  
  Source code used for data processing and visualization generation.

- **`/benchmark_data/`**  
  Raw benchmark datasets used in the analysis.

- **`/model_data/`**  
  Model Dataset.

- **`/results/`**  
  Generated visualizations and analysis outputs.  
  These files are reproducible and can be regenerated by running the scripts in `/src` or the notebook below.

- **`results.ipynb`**  
  Jupyter notebook that reproduces all visualizations in the project.

- **`/docs/`**  
  Project documentation, including `project.md`.

- **`Presentation pdf`**  
  Presentation slides summarizing the project and results.


## How to run

### Prerequisites
* Python 3.11+
* Any conda environment or venv
* Dependencies: `pandas`, `numpy`, `matplotlib`, `seaborn`, `scipy`

### Execution
1.  Clone the repository.
2.  Open and run `results.ipynb`. (Or run individual files in `/src`)

Note: For more information about the results, please consult `docs/project.md`.
