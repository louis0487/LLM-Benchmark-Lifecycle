# The Lifecycle and Clustering of LLM Benchmarks – A Data Driven Analysis
## Project Information
For project information, technical approach and results, please see `docs/project.md`. 

## How to run

### Prerequisites
* Python 3.11+
* Any conda environment or usual venv
* Dependencies: `pandas`, `numpy`, `matplotlib`, `seaborn`, `scipy`

### Execution
1.  Clone the repository.
2.  Open and run `analysis.ipynb`:
    * **Cell 1**: Data Loading and Preprocessing.
    * **Cell 2**: Logistic Curve fitting and visualization for specific benchmarks.
    * **Cell 3**: Clustered Heatmap generation for correlation analysis.
